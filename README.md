# R Data Explorer – Module #3

## Overview
This project is a menu‑driven R program that allows users to load a CSV file and perform basic data exploration tasks. The tool includes summary statistics, data filtering, random number generation, histogram creation, and saving summaries to a file. This project was completed as part of Module #3 in CSE 310.

---

## Features
- Load and read CSV data
- View summary statistics for numeric columns
- Filter data based on a column and threshold
- Generate random numbers
- Create histograms for selected columns
- Save summary statistics to a text file
- Menu‑driven interface for easy navigation

---

## How to Run the Program
1. Open R or RStudio.
2. Place `RDataExplorer.R` and your CSV file in the same folder.
3. Run the script:
